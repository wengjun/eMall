- 想建立项目整体认知，先读系统总览、架构设计和设计深度说明。
- 想准备 Amazon L6/Senior SDE 面试，重点读 L6 面试导向、概念到实现补强和面试表达。

- 想做生产运行，重点读中间件平台、安全可观测性、测试治理和生产就绪。
