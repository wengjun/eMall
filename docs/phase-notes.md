# 阶段说明

本文压缩记录 P4 到 P8 阶段的模块实现重点和生产扩展边界。

## P4：平台治理

涉及模块：

- `identity`
- `risk`
- `operations`
- `openapi`

完成内容：

- 身份、会话、权限、服务客户端和商家子账号。
- 风控规则、设备信誉、风险事件和反作弊决策。
- 审批、运维任务、合规证据和安全事件。
- 开放平台应用、签名校验、配额和 Webhook。

生产扩展：

- SSO、MFA、细粒度权限。
- 规则引擎和风控模型。
- 内部操作控制台。
- 商家开放平台开发者门户。

## P5：业务增长平台

涉及模块：

- `catalog`
- `promotion`
- `experiment`
- `advertising`

完成内容：

- 类目、属性、品牌授权、SPU/SKU 审核和发布。
- 活动生命周期、叠加优惠和预算消耗。
- 实验流量分配、互斥组、护栏和报告。
- 广告活动、素材、关键词定向、排序和扣费。

生产扩展：

- 商品治理后台。
- 更复杂的促销规则引擎。
- A/B 实验统计显著性。
- 广告归因和反作弊。

## P6：供应、财务和客服

涉及模块：

- `supply-chain`
- `finance`
- `customer-service`
- `forecasting`

完成内容：

- 入库、上架、调拨、运单、配送异常和签收凭证。
- 账户、流水、冻结资金、结算、发票、清分和拒付。
- 工单、分派、仲裁、补偿、知识库和服务评价。
- 需求信号、预测、补货计划、履约产能和风险汇总。

生产扩展：

- WMS/TMS 深度集成。
- 财务合规和税务规则。
- 客服工作台和质检。
- 采购、仓配和履约预测模型。

## P7：数据、AI 和分析

涉及模块：

- `event-platform`
- `data-warehouse`
- `intelligence`
- `analytics`

完成内容：

- 事件 schema、埋点采集、offset 和指标物化。
- ODS、DWD、DWS、ADS 数据集、分区、质量检查和血缘。
- 用户画像、商品画像、在线特征、模型部署和 AI 决策。
- 指标治理、BI 看板、异常、授权和隐私请求。

生产扩展：

- 实时数仓和流批一体。
- 特征平台和模型训练平台。
- 数据质量监控。
- 隐私合规和访问审计。

## P8：生产规模控制面

涉及模块：

- `traffic`
- `reliability`
- `release`
- `platform-ops`

完成内容：

- 单元、分片路由、流量切换和隔离。
- 容量演练、SLO、混沌审批和上线门禁。
- 功能开关、灰度、Topic 治理和回放计划。
- 备份、数据库操作、FinOps 和安全操作。

生产扩展：

- 多活流量调度。
- 自动化容量认证。
- 发布平台和自动回滚。
- 平台运维控制台。
